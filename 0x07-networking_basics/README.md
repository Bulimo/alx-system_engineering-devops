# 0x07-networking_basics

## Objectives

    . Understanding networking and network protocols
    . Know the different types of networks (LAN, WAN, WLAN, MAN ...)
    . Understand the OSI Model

## Environment Requirements

    . Allowed editors: vi, vim, emacs
    . All files will be interpreted on Ubuntu 20.04 LTS
    . All files should end with a new line
    . A README.md file, at the root of the folder of the project
    . All Bash script files must be executable
    . Bash script must pass Shellcheck without any error
    . The first line of all Bash scripts should be exactly #!/usr/bin/env bash
    . The second line of all Bash scripts should be a comment explaining what is the script doing

## Files in the folder

    . 4-TCP_and_UDP_ports - script to print the active ports in a PC
    . 5-is_the_host_on_the_network - script to print an IP address passed as an argument
